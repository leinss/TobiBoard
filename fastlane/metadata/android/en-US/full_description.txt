Everything runs on your device. Cloud is optional.

TobiBoard is a fork of HeliBoard — the best privacy-respecting Android keyboard — with the three things it was missing: on-device voice-to-text, on-device text fix, and a proper clipboard manager. All run locally by default. No account. No API key required. Cloud providers are available if you want more powerful models, but you never need one.

<b>On-device Voice-to-Text</b>
Long-press Return, tap the mic, speak. Your words land as polished text. Uses an on-device Whisper model by default — audio never leaves your phone.
<ul>
  <li>Works offline, no key needed</li>
  <li>Cloud models (OpenRouter, PayPerQ) available if you want them — bring your own key</li>
  <li>Custom prompt or vocabulary so it nails names and jargon</li>
</ul>

<b>On-device Text Fix</b>
Select rough text, long-press Return, hit Fix. Back as clean text in the same language. Runs locally by default — nothing uploaded, no key needed.
<ul>
  <li>Works offline, no key needed</li>
  <li>Any language, cleans typos and awkward phrasing, adjusts tone</li>
  <li>Review the rewrite before replacing the original</li>
</ul>

<b>Clipboard history — stays on your device</b>
Long-press the clipboard icon to browse everything you've copied. See when it was copied and how often you paste it.
<ul>
  <li>Pin entries you paste often so they're always at the top</li>
  <li>Add labels to find things later</li>
  <li>Search, manage, and delete from Settings — never uploaded or backed up to the cloud</li>
</ul>

<b>When you do go cloud</b>
TobiBoard requests OpenRouter zero-data-retention endpoints by default — your audio and text aren't logged or used for training when the model supports it. API keys are encrypted with the Android Keystore and never written to logs.

<b>All the HeliBoard features you already know</b>
<ul>
  <li>Dictionaries, spell check, multilingual typing</li>
  <li>Themes, dynamic colors, background image, day/night</li>
  <li>Custom layouts, number row, one-handed mode, split keyboard</li>
  <li>Backup and restore settings and custom assets</li>
</ul>

TobiBoard installs side-by-side with HeliBoard, so you can keep both.
